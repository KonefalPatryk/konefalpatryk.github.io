Motion Capture Test (Personal Pandemic Project 2020)

I wanted to learn how games work and played around with game files using various programs. I learned how to connect a Kinect v1 & Kinect v2 to the computer and how to use them for motion capture and photogrammetry, then used a character model from the game BattleForge (2009) (now called Skylords Reborn) to learn about rigging and weight painting, combining it all together for a short video.

The recording demonstrates the use of a single Kinect v2 sensor, resulting in noticeable jitter in the animation due to its limitation of tracking from only one direction.

Upon reflection, I have learned how to better optimize an animation by adjusting the recording environment and ensuring it's set under the appropriate conditions for the best motion tracking quality.

Key things I've learned:

  • Rigging
  • Weight Painting
  • Motion Capture
  • Photogrammetry
